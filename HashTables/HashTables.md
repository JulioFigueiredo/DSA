# Hash Tables

Uma Hash Table é uma estrutura de dados que mapeia chaves para valores. Ela utiliza uma função de hash para calcular um índice em um array, onde o valor correspondente será armazenado.

## Por que usar?

A principal vantagem é a velocidade. Em um cenário ideal, as operações de busca, inserção e remoção têm complexidade de tempo constante: $O(1)$.
